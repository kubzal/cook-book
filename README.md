# Git commands
 - `git init` - initialize git repository
 - `git status` - current status of your repository
 - `git add` - adds files to staging
 - `git commit` - commits files (create a saved snapshot of your files)
 - `git diff` - compares files current state vs last commit
 - `git log` - shows commits history

## Remote repositories
 - `git remote add origin [url]` - connect to a remote repository, origin is the conventional name for remote repositories
 - `git remote -V` - check the url of _origin_ (remote repository)
 - `git clone` - "cloning" remote repository to my local machine (we create a local copy with link to a remote repository)
